# picotouch_synth

Thin captouch-based synth platform for Raspberry Pi Pico

<img width=500 src="./docs/picotouch_synth_flip_case_render1.jpg"/>

Features:
* PWM Audio out
* TRS-A MIDI In
* TRS-A MIDI Out
* USB MIDI In/Out
* 19 RGB LEDs
* 22 touch pads (17 notes, 5 modifiers)
